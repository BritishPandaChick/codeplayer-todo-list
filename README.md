# CodePlayer To Do List
Codeplayer.com to do list project with three items that you can mark as complete or incomplete

## Tech Stack
- HTML
- CSS

## Instructions
1. Clone or download zip file. 
2. Open index.html file in your browser.
3. The to-do list will be in the center of the screen. Completed items will have check marks and be crossed off. Click on items to mark them as complete. You can also click on completed items to remove the check mark.
